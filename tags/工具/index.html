一、窗体绘制的常用设置

窗体的设置应当在窗体构造函数中InitializeComponent()方法前执行

public frmMain()
{
    this.StartPosition = FormStartPosition.CenterScreen;//窗体居中显示  
    this.MaximizeBox = false;//不显示最大化按钮 
    this.FormBorderStyle = FormBorderStyle.FixedSingle;//禁止放大缩小 
    InitializeComponent();
}
